Python: projektowanie gier 3j

Opis

Ten skrypt to szkielet prostej gry tekstowej, w której gracz tworzy własną postać, wybiera dwie umiejętności z listy, a następnie próbuje wydostać się z trójpokojowego labiryntu. Program pokazuje, jak łączyć w jednym projekcie kilka ważnych elementów Pythona: funkcje, listy, słowniki (mapujące nazwy pokoi na opisy i wyjścia), pętle while oraz obsługę wejścia użytkownika za pomocą input(). Struktura labiryntu jest zdefiniowana w słowniku LABIRYNT, dzięki czemu uczeń może łatwo zmieniać opisy i połączenia między pokojami, nie dotykając logiki gry. To dobry punkt wyjścia do dalszych eksperymentów – dodawania nowych pomieszczeń, warunków, a nawet prostych zagadek zależnych od wybranych umiejętności bohatera.

Kod gry



import os
import time

CLEAR = "cls" if os.name == "nt" else "clear"
skills = ['wytwarzanie', 'walka', 'perswazja', 'zręczność']

# =========================================
# 1. KOD NAUCZYCIELA – TEGO NIE RUSZAMY
# =========================================

def tworzeniePostaci(umiejetnosci):
    postac = []
    skile = []
    liczba_umiejetnosci = 2

    nazwa = input('Nazwa postaci: ')
    postac.append(nazwa)

    os.system(CLEAR)
    print('Wybierz z listy 2 umiejętności!')
    print('Dostępne umiejętności')
    print('#####################')
    for skil in umiejetnosci:
        print(skil)
    print('#####################')
    input('Dalej - Enter')

    for skil in umiejetnosci:
        if liczba_umiejetnosci == 0:
            break
        os.system(CLEAR)
        print('Twój wybór to:')
        print(skil)
        wybieram = input('tak/nie: ').strip().lower()
        if wybieram == 'tak':
            if skil in skile:
                print('Masz już tę umiejętność. Wybierz inną.')
                time.sleep(1)
                continue
            skile.append(skil)
            liczba_umiejetnosci -= 1
            print(f'Wybrałeś: {skil}. Zostało: {liczba_umiejetnosci}')
            time.sleep(1)
        elif wybieram == 'nie':
            continue
        else:
            print('Wpisz "tak" albo "nie".')
            time.sleep(1)

    postac.append(skile)
    return postac

def prezentacjaPostaci(postac):
    os.system(CLEAR)
    nazwa, skile = postac
    umiejetnosci_txt = ", ".join(skile) if skile else "— brak —"
    print("########################")
    print(f"Witaj {nazwa}")
    print("Twoje umiejętności:")
    print(umiejetnosci_txt)
    print("########################")

def pierwszaPlansza():
    os.system(CLEAR)
    print("Znajdujesz się w labiryncie")
    print("Twoim zadaniem jest się")
    print("wydostać z labiryntu")

# =========================================
# 2. SZABLON DLA UCZNIA – TU WYPEŁNIASZ
#    dane z karty pracy (3 pokoje)
# =========================================

LABIRYNT = {
    # START
    "start": {
        "opis": "TU WPISZ OPIS STARTU (z karty pracy)",
        "wyjscia": {
            # w/s/a/d -> nazwa innego pokoju
            "w": "srodek"   # PRZYKŁAD, UCZEŃ MOŻE ZMIENIĆ
        }
    },

    # ŚRODEK
    "srodek": {
        "opis": "TU WPISZ OPIS ŚRODKA (z karty pracy)",
        "wyjscia": {
            "s": "start",
            "w": "wyjscie"
        }
    },

    # WYJŚCIE
    "wyjscie": {
        "opis": "TU WPISZ OPIS WYJŚCIA (z karty pracy)",
        "wyjscia": {
            "s": "srodek"
        }
    }
}

def gra_w_labirynt(postac):
    aktualne_miejsce = "start"  # nazwa musi pasować do LABIRYNT

    while True:
        os.system(CLEAR)
        nazwa, skile = postac

        print(f"Bohater: {nazwa}")
        if skile:
            print("Umiejętności:", ", ".join(skile))
        else:
            print("Umiejętności: brak")
        print("====================================")

        dane = LABIRYNT[aktualne_miejsce]
        print(dane["opis"])
        print("")

        if aktualne_miejsce == "wyjscie":
            print("\nDotarłeś do wyjścia z labiryntu!")
            input("Naciśnij ENTER, aby zakończyć wędrówkę...")
            break

        wyjscia = dane["wyjscia"]
        print("Możesz iść:")
        for kierunek in wyjscia:
            nazwa_kierunku = {
                "w": "góra",
                "s": "dół",
                "a": "lewo",
                "d": "prawo"
            }[kierunek]
            print(f"  {kierunek} ({nazwa_kierunku})")

        print("  q (wyjście z gry)")
        ruch = input("Twój ruch: ").strip().lower()

        if ruch == "q":
            print("Rezygnujesz z wędrówki po labiryncie.")
            time.sleep(1)
            break

        if ruch in wyjscia:
            aktualne_miejsce = wyjscia[ruch]
        else:
            print("Nie możesz iść w tym kierunku.")
            time.sleep(1)

# =========================================
# 3. START PROGRAMU – TEGO NIE RUSZAMY
# =========================================

bohater = tworzeniePostaci(skills)
prezentacjaPostaci(bohater)
input('Enter')
pierwszaPlansza()
input("Naciśnij ENTER, aby wejść do labiryntu...")
gra_w_labirynt(bohater)

Zadania do wykonania

Sekcja do samodzielnej pracy – zaprojektuj swój własny labirynt i wpisz go do programu.
  1. Na kartce (albo w zeszycie) narysuj prosty labirynt i zaznacz na nim 3 miejsca: START, ŚRODEK (ciekawe miejsce) i WYJŚCIE. Zapisz krótko, gdzie one są (np. „przy wejściu po lewej stronie”, „na środku planszy”).
  2. Wymyśl dla każdego z tych trzech miejsc nazwę „kodową” do Pythona – tylko małe litery, bez spacji i polskich znaków, możesz użyć _ i cyfr, np. wejscie_start, sala_pajakow, wyjscie1.
  3. Napisz krótkie opisy dla gracza:
    • co widzi bohater na STARcie,
    • co to za miejsce w ŚRODKU (jak wygląda, co czuć / słychać),
    • jak wygląda WYJŚCIE z labiryntu.
    Każdy opis może mieć 1–3 zdania.
  4. Zastanów się, w które strony można iść z każdego miejsca (w – góra, s – dół, a – lewo, d – prawo). Dla każdego z trzech pokoi zapisz, dokąd prowadzi dany kierunek. Jeśli nie ma przejścia, wpisz „X”.
  5. Otwórz kod gry i znajdź część „SZABLON DLA UCZNIA – LABIRYNT” (słownik LABIRYNT). Zastąp domyślne nazwy "start", "srodek", "wyjscie" swoimi nazwami „kodowymi” oraz wstaw własne opisy w miejsce tekstów "TU WPISZ OPIS ...".
  6. W każdym pokoju uzupełnij słownik "wyjscia" tak, aby odpowiadał Twojemu rysunkowi: dla w, s, a, d wpisz nazwę pokoju, do którego prowadzi przejście (albo usuń kierunek, jeśli wpisałeś „X”). Uruchom program i sprawdź, czy da się przejść od STARTu do WYJŚCIA.
